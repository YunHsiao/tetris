--------------------------------------------------------------------------------------
N 		新游戏
S 		保存为自定义文本格式文件(tst)
Shift+S		保存为二进制文件(tsf)
Ctrl+S 		保存为xml文件(xml)
L 		读取
Q 		退出
ESC 		暂停
--------------------------------------------------------------------------------------
←		向左移动
→		向右移动
↑或Space	旋转
↓		加速下落
双击↓		直接落位
--------------------------------------------------------------------------------------
启动时按下Shift	使用OpenGL渲染
启动时按下Ctrl	使用GDI渲染
正常启动		使用Direct3D渲染
--------------------------------------------------------------------------------------
Direct3D模式下:
F7		使用ID3DXFont渲染字体
F8		使用freetype渲染字体
--------------------------------------------------------------------------------------
