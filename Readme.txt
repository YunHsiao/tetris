--------------------------------------------------------------------------------------
N 		新游戏
S 		保存为自定义文本格式文件(tst)
Shift+S		保存为二进制文件(tsf)
Ctrl+S 		保存为xml文件(xml)
L 		读取
Q 		退出
ESC 		暂停
--------------------------------------------------------------------------------------
←		向右移动
→		向左移动
↑或Space	旋转
↓		加速下落
--------------------------------------------------------------------------------------
启动时按下Shift	使用OpenGL渲染
启动时按下Ctrl	使用GDI渲染
正常启动		使用Direct3D渲染
